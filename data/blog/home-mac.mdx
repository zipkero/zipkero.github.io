## 설치

### 필수

- 크롬
- xcode command line tools

```bash
xcode-select --install
```

- homebrew [https://brew.sh/index_ko]
- node

```bash
brew install node
```

- go

```bash
brew install go
```

- git

```bash
brew install -s git
```

- iterm2

```bash
brew install --cask iterm2
```

- Visual Studio Code [https://code.visualstudio.com/]
- Ridibooks
- Jetbrain Toolbox
    - Goland
    - Webstorm
    - Datagrip (선택)
- CleanMyMac
- KakaoTalk
- Docker

### 선택

- Pocket
- gSwitch (m계열은 필요없음)

```bash
brew install --cask gswitch
```

- Dropbox
- Notion
- Figma
- PostMan
- Slack & Discord
- Virtual Box

### Utility

```bash
# tree
brew install tree
# 쿠버네티스 사용하면
brew install kubernetes-cli
# 테라폼 사용하면
brew install terraform
# virtual box 와 vagrant 사용하면
brew install vagrant
```

## 설정

- ssh-key

```bash
ssh-keygen
```

- ssh 로 github push 가 되지 않는 경우 .ssh 폴더에 config 파일 생성 후 아래와 같이 기입

```
Host github.com
    Hostname ssh.github.com
    Port 443
```

- 터미널 단축키 .zshrc

```
alias k="kubectl"
alias ll="ls -al"
```
