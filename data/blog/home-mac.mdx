## 설치

### 필수

- 크롬
- xcode command line tools

```bash
xcode-select --install
```

- homebrew [https://brew.sh/index_ko]
- node

```bash
brew install node
```

- go

```bash
brew install go
```

- git

```bash
brew install -s git
```

- iterm2

```bash
brew install --cask iterm2
```

- Visual Studio Code [https://code.visualstudio.com/]

```bash
brew install --cask visual-studio-code
```

- Jetbrain Toolbox
  - Goland
  - Webstorm
  - Datagrip (선택)
- CleanMyMac or AppCleaner + RunCat
- KakaoTalk
- Docker
- etc

```bash
# tree
brew install tree
# 쿠버네티스 사용하면
brew install kubernetes-cli
# 테라폼 사용하면
brew install terraform
# virtual box 와 vagrant 사용하면
brew install vagrant
```

### 선택

- Pocket
- gSwitch (m계열은 필요없음)

```bash
brew install --cask gswitch
```

- Dropbox
- Ridibooks
- Notion
- Figma
- PostMan
- Slack & Discord
- Parelles or Virtual Box
  - 애플 실리콘 계열은 현재까지는 Virtual Box, VMWare 에서 완벽히 지원이 되지 않는다.

## 설정

- ssh-key

```bash
ssh-keygen
```

- ssh 로 github push 가 되지 않는 경우 .ssh 폴더에 config 파일 생성 후 아래와 같이 기입

```
Host github.com
    Hostname ssh.github.com
    Port 443
```

- 터미널 단축키 .zshrc

```
alias k="kubectl"
alias ll="ls -al"
```

- 맥북 대기상태 전력소모량 줄이기(ventura 부터는 사라진 것으로 보임)

```bash
sudo pmset standbydelaylow 900 standbydelayhigh 1800
# 실리콘 맥북은 기본적으로 0으로 설정되어 있는 듯 하다.
# sudo pmset standbydelaylow 0 standbydelayhigh 0
```
